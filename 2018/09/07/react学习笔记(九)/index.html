refs:是react中专门提供通过操作DOM来实现需求的方式，它是一个对象，存储了当前组件中所有设置了ref属性的元素(元素ref属性值是啥,refs中存储的元素的属性名就是啥)

在react组件中：

  • 2、基于REF操作DOM实现视图更新的，叫做”非受控组件”

=> 真实项目中，建议多使用”受控组件”
