数组常见方法的使用及其封装

发表于 2018-08-26 | 分类于 数组
字数统计: 5,146 | 阅读时长 ≈ 23

push

  • 向数组末尾新增一项；
  • 参数是新增的那一项;可以传多个；
  • 新数组的数组成员的个数；
  • 原有数组发生改变；

pop

  • 删除数组的最后一项
  • 不需要传参数
  • 被删除的那一项
  • 原有数组发生改变

unshift

  • 向数组开头新增一项；
  • 需要传参数
  • 新数组的数组成员个数
  • 原有数组发生改变；

shift

  • 删除数组的第一项
  • 不需要传参
  • 被删除的那一项
  • 原有数组的发生改变；

slice

  • 数组的截取
  • slice(m,n): 从数组索引m开始，截取到索引n，但是不包含n;[前包后不包]
  • slice(m) : 从索引m开始，截取到末尾；
  • slice():数组的克隆 slice(0);
  • 索引负数： 让当前length+负数；
  • 返回值是截取的数组
  • 原有数组不发生改变；

分析：

splice

  • 删除数组中的某几项
  • splice(m,n): 从索引开始，删除n个
  • splice(m) : 从索引m开始删除到末尾；
  • splice(0):
  • splice(m,x,n);替换从索引m开始，删除x个，用n替换；
  • 返回值；删除那几项，并且以数组返回
  • 原有数组发生改变；

分析

sort

  • 数组排序
  • 参数
  • sort() : 只能数组成员是相同位数的数组
  • sort(function(a,b){return a-b}) 从小到大排序
  • sort(function(a,b){return b-a})从大到小
  • 是排序之后的数组
  • 原有数组发生改变；

reverse

  • 将数组颠倒过来
  • 不需要传参数
  • 数组成员顺序倒过来之后的数组
  • 原有数组发生改变；

concat

  • 数组的拼接
  • 不传参数： 数组的克隆
  • 传参数，（数组、每一项）；把传入的实参拼接新的数组中；
  • 拼接之后的新数组
  • 原有数组不发生改变；

join

  • 把数组成员按照特定的字符连接成一个字符串；
  • 不传参数,会默认按照逗号分开
  • 传参数，(特定的字符)
  • 拼接之后的字符串
  • 原有数组不发生改变

indexOf

  • 检测数组成员在数组中第一次出现的索引位置； 判断当前项是否在数组中存在；如果不存在，返回-1；
  • 需要参数
  • 返回在数组中第一次出现的索引；
  • 原有数组不发生改变

lastIndexOf

  • 检测数组成员在数组中最后一次出现的索引位置； 判断当前项是否在数组中存在；如果不存在，返回-1；
  • 需要参数
  • 返回在数组中最后一次出现的索引；
  • 原有数组不发生改变；

map

  • 遍历数组和映射
  • 需要参数
  • 映射之后的数组
  • 原有数组不发生改变

forEach

  • 遍历数组;没有返回值；
  • 需要参数
  • 返回值是undefined
  • 原有数组不发生改变

toString

  • 转字符串
  • 不需要参数
  • 返回一个去了中括号之后的字符串
  • 原有数组不变

filter

过滤;把符合条件的放进新数组中；原有数组不发生改变

find

查找；从所到右一次进行查找；找到符合条件的第一项，并且把这一项返回；结束整个函数

some

返回一个布尔值；如果有符合条件的返回true，如果没有找到符合条件的，返回是false

every

返回一个布尔值；true或false；必须要求每一项都返回true，整体结果返回true；只要有一个是false，整体结果就是false

includes

包含；检测是否包含某一项；如果包含返回true，不包含返回false；

reduce

收敛；求和
